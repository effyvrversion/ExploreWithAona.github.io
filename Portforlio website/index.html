My Portforlios

Python: Video Games Analysis

This project involves data cleaning and manipulating in Python. To get an overview of game industry in the past four decades, I applied Python libraries(Pandas, Seaborn, Matplotlib) and made an interactive dashboard in Tableau.

Bank Customer Analysis

This project aims to retrieve general customer profiles and key elements affecting exited cases from customer data. The outcome can enhance customer analysis to facilitate solutions to minimize customer exit rates.

Data Extraction and Visualization: Artwork metadata

The project expects to find connection between digital artwork metadata and color patterns that may shed light on broader artistic or cultural movements. The extraction of source data combines web scraping and API access. Here I made an overview of Rijksmuseum metadata and detailed analysis of two representative artists-Vincent van Gogh and Rembrandt van Rijn.

SQL: Mondial Data

Mondial is a set of tables that cover information of country, border, economy, population and nature resourcess. I select this database to explore the state of nature resource sustainability and its relation with economy.

Machine Learning:Image Clustering & Semantic Search

This is a test dataset with 500 images by Van Gogh. I insert the CLIP embedding to cluster the images and achieve search funtion based on image features(color and object).

Space X Falcon 9 First Stage Landing Prediction

This task creates a machine learning pipeline to predict if the first stage of SpaceX Falcon 9 will land given the data. The process applies three machine learning methods: SVM, Classification Trees and Logistic Regression.
